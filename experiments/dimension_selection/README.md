# Dimension Selection

Dieser Ordner enthält Skripte zur Untersuchung der effektiven Dimensionswahl,
insbesondere zwei-skalige Kernel und Parameter-Scans.

Wichtige Dateien:
- `DimensionsHeatmap*.py`: Heatmap- und Dimension-Scan-Skripte
- `plotD.py`, `plotDgpu.py`: Visualisierung der Ergebnisdaten

Ziel: Neue Experimente sollten reproduzierbare Eingabeparameter und
klar dokumentierte Ausgabepfade verwenden.
