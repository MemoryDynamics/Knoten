# Propagation Speed

Skripte für die Analyse von Light-Cone-Struktur, retardierten Effekten und
propagation speed. Diese Dateien sind eng mit dem Paper II-Thema verbunden.

Wichtige Dateien:
- `PaperII3D_4Plots.py`, `PaperII3D_5Plots*.py`: Plot-Skripte für Paper II

Ziel: Neue Reproduktionen sollten Daten aus `data/raw/` generieren und in
`data/processed/` ablegen.
